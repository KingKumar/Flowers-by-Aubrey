# AGENTS.md

## Project Overview
This is a Next.js (App Router) + Tailwind CSS website for a floral design business.

The goal is to create a clean, elegant, conversion-focused marketing site that showcases floral arrangements and drives customer inquiries.

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Development Rules
- Keep components simple and readable
- Do NOT introduce unnecessary libraries or frameworks
- Avoid over-engineering
- Prefer functional components
- Use Tailwind for styling

## Design Guidelines
- Minimal, elegant, soft, premium floral aesthetic
- Lots of whitespace
- Soft color palette (cream, blush, sage, muted tones)
- Consistent spacing and typography
- Mobile-first design

## Homepage Structure
1. Hero section (image + name + tagline)
2. Gallery (bouquets)
3. Services (custom, events, delivery)
4. About (short, warm)
5. Call-to-action (contact/order)
6. Footer

## Images
- Use high-quality floral placeholder images if needed
- Use Next.js Image component

## Components
- Keep components modular (Hero, Gallery, etc.)
- Create a /components folder if needed

## Performance
- Prefer server components
- Avoid unnecessary client-side state

## Do NOT add
- Authentication
- Database
- Complex animations
- Extra libraries unless asked

## Tone
- Warm, natural, slightly luxurious
- Not corporate, not techy

## Priority
1. Clean UI
2. Clear messaging
3. Fast load
4. Mobile experience
